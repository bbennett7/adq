# askdumbquestions.ai

A personal AI-education publication. One foundational AI question, one answer, every weekday.

## This is NOT the Next.js you know

This project uses Next.js 16 (App Router). APIs, conventions, and file structure may differ from training data. Read `node_modules/next/dist/docs/` before writing any code. Heed deprecation notices.

## Stack

- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS + CSS custom properties (see `globals.css`)
- **Fonts:** Libre Franklin (wordmark), Newsreader (display/body), Inter Tight (UI), DM Mono (metadata)
- **Deploy:** Vercel
- **Package manager:** pnpm — always use `pnpm`, never `npm` or `yarn`

## Content model

1. **Daily Q&A** — one AI question + answer, weekdays. Generated by a Vercel Cron agent (Claude API), owner picks and publishes.
2. **Field Notes** — occasional long-form essays, manually authored.

|---|---|---|---|

### `POST /api/revalidate`

Triggers on-demand ISR revalidation. Used by the Vercel Cron agent after publishing a new question.

**Auth:** Request header `x-revalidation-secret` must match `REVALIDATION_SECRET` env var.

**Body:**
```json
{ "questionNumbers": [142] }
```

**Behavior:** Always revalidates the `questions` tag (recent questions list). Also revalidates `question-{n}` for each number in `questionNumbers`.

**Response:** `{ "revalidated": true }` on success; `401` if secret missing or wrong.
